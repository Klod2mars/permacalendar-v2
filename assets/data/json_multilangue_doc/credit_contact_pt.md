## Sobre

Este projeto é desenvolvido de forma independente, com uma abordagem sóbria e progressiva, atenta ao longo prazo e à qualidade dos usos.

Nenhum dado pessoal é coletado através desta página. Nenhum formulário, nenhum sistema de rastreamento ou armazenamento de mensagens está integrado ao aplicativo.

Qualquer troca é feita voluntariamente, fora do aplicativo, através dos contatos indicados acima.
